# 나만의 개시판

Mybatis, JSP를 연습하기 위해 만든 게시판 입니다.
개발 인원 : 1명
개발 기간 : 2023년 8월 23일 ~ ...

---

## 기술 스택
- Java 11
- Spring boot
- Spring Security
- Mybatis
- Mariadb
- JSP, jquery, ajax

## 주요 기능
- 게시글 CRUD
- Spring Security를 통한 로그인 관리
- 회원 정보 수정(아이디, 비밀번호, 닉네임, 전화번호, 이메일), 삭제
- 게시글 Search (개발 중)
- 추천, 비추천 등록 (개발 중)
- 조회 수 증가(개발 중)
